Getting Started - Instructions

How to Use This Tool

IMPORTANT: Start Here!

Before making any room assignments, you MUST first populate the Data Table with your course and room information. The assignment tools depend on this data being entered first.

Step-by-Step Workflow:

  1. Data Table Section: Start by entering all your courses and room data
  2. Classroom Management: Configure available rooms and RMU statuses
  3. Course Assignments: Assign rooms to courses using the dropdown interface
  4. Room Swap Tool: Use to exchange room assignments between courses
  5. Generate Report: Create final PDF reports for stakeholders

Pro Tips:

  • Save your work frequently using "Save to File"
  • Use the "Base Calculations" dropdown to switch between attendance years
  • Check the Assignment Summary for capacity issues
  • Use the Room Swap Tool to quickly exchange assignments
  • Export to CSV for external analysis

Data Table Setup (Required First Step):

Adding Courses: Click "Add New Course" to create course entries with attendance data

Adding Rooms: Use "Classroom Management" section to add rooms with capacity and details

Editing Data: Click any cell in the Data Table to edit values directly

Importing Data: Use "Load from File" if you have a previously saved configuration

Common Issues to Avoid:

  • Don't try to assign rooms before adding course and room data
  • Make sure room capacities are entered correctly for accurate calculations
  • Check RMU statuses in Classroom Management before finalizing assignments
  • Use "Refresh Table" if data appears out of sync between sections
  • Save your work before generating reports or making major changes

Quick Reference:

Data Entry:
• Click cells to edit
• Tab to move between cells
• Enter to save changes
Room Management:
• Check boxes for availability
• Set RMU status for each room
• View current assignments
File Management:
• Save early and often
• Use descriptive file names
• Export reports as PDF
